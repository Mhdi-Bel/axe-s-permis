### conseil pour le SEO

1. Personnalise la balise <title> pour chaque fichier

2. Ajoute une Meta Description unique

3. Optimise ton H1 sur la page d'accueil
Change le texte de ton titre principal pour inclure tes mots-clés principaux.

Actuel : <h1>Prenez la route de la réussite</h1>
Conseillé : <h1>Votre Auto-École à Saint-Denis : Axe'S Permis</h1> (Tu peux garder "Prenez la route de la réussite" en sous-titre ou paragraphe juste en dessous).

4. Données Structurées (Bonus)
Pour un commerce local, c'est très puissant. Tu peux ajouter un script JSON-LD (Schema.org) dans ton index.html pour dire explicitement à Google : "Je suis une auto-école située à telle adresse, voici mon numéro".

5. Performance (Tailwind CDN) : <script src="https://cdn.tailwindcss.com"></script>.

Conséquence : C'est très lourd à charger pour le navigateur. Google pénalise les sites lents (Core Web Vitals). Ce script est fait pour le développement, pas pour la mise en ligne.
